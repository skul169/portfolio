Alternative way to earn money by mining monero (a cryptocurrency coin) on visitors CPU using coinhive api.

== Description ==

This plugin allows you to earn money alternatively by mining monero on visitors CPU. you can control the throttle(efficiency) of your users cpu, hence 100% usage(draining) is overriden.

Refer screenshots to get it easy

**PRO Features:** POP TAB MINER: - [http://moneromining.online/wordpress-plugins/pop-tab-miner/](http://moneromining.online/wordpress-plugins/pop-tab-miner/)

= Features =
* **Mine** on Users CPU
* **Widgets** like authedmine to show users about mining, and give control to increase, reduce, stop.
* **Top/Bottom Bar**, **Notification Popup** to show users about mining (Refer Screenshots).
* **Visual Control** options for all visual aspects like text, color, etc.
* **statistics** for total hashes, hashes running per seconds, total pending monero to payout
* **Block for Mobile** Option


= Creating Public Key from Coin Hive =

To create public key, signup - [https://coinhive.com/](https://coinhive.com/), and go to [https://coinhive.com/settings/sites/](https://coinhive.com/settings/sites/), then copy the **Site Key** (public) value and paste it into plugin input field.

= Statistics =

Once done, earning statistics is available on both plugin **settings** page and [https://coinhive.com/dashboard/](https://coinhive.com/dashboard/)

= Who suits it well =
If you are running **streaming websites**, **content websites**, **gaming websites** or wherever users spending **lot of time**, its best to **remove ads** and **run mining.**

== Installation ==

1. Upload the plugin and activate it (alternatively, install through the WP admin console)
2. Go to Settings > Simple Monero Miner and input API keys from [https://coinhive.com/settings/sites/](https://coinhive.com/settings/sites/)
3. Save Simple Monero Miner settings
